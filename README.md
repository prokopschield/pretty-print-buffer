# pretty-print-buffer

Shows you the contents of a buffer using pretty colors.

### Installation

```sh
yarn global add pretty-print-buffer
```

### CLI usage

```sh
my_command | pretty-print-buffer
```

### JS usage

```js
const pretty = require('pretty-print-buffer');

pretty(my_buffer);
```

### TS usage

```ts
import pretty from 'pretty-print-buffer';

pretty(my_buffer);
```
